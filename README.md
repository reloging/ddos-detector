# ML DDoS Detector

Система обнаружения DDoS-атак на основе машинного обучения.

---

## Описание

Проект реализует интеллектуальную систему защиты веб-сервера от DDoS-атак.
Детектор анализирует сетевой трафик в реальном времени и использует модель
RandomForest для классификации запросов на нормальные и атакующие.

```
+----------------+     +-------------+     +----------------+
|   Атакующий    | --> |   Сервер    | --> |   Детектор     |
|                |     |   Flask     |     |   (ML модель)  |
+----------------+     +-------------+     +----------------+
                                             |
                                             v
                                    +----------------+
                                    |   Firewall     |
                                    |   блокировка   |
                                    +----------------+
```

---

## Принцип работы

1. **Сбор данных**: Сервер логирует каждый запрос в stdout
2. **Агрегация статистики**: Детектор собирает метрики по каждому IP
3. **Извлечение признаков**: Формируется вектор из 70 признаков (скорость, размер пакетов, флаги TCP)
4. **ML-классификация**: RandomForest предсказывает вероятность атаки
5. **Блокировка**: При превышении порога IP блокируется через Windows Firewall
